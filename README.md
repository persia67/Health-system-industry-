# سامانه جامع مدیریت سلامت شغلی (Occupational Health System - OHS)

سامانه جامع سلامت شغلی (OHS) یک پلتفرم دسکتاپ و وب قدرتمند، مدرن و با کلاس کاری بالا است که برای خودکارسازی فرآیند پایش سلامت کارکنان، مدیریت پرونده‌های طب کار، ثبت آزمایش‌های پیراپزشکی و ارائه گزارش‌های دقیق مدیریتی طراحی شده است. این سیستم با هدف بهبود کارایی فرآیندها، حفظ امنیت داده‌های حساس پزشکی و ارائه تحلیل‌های هوشمند به پزشکان و کارشناسان HSE توسعه یافته است.

---

## 🚀 ویژگی‌های کلیدی (پیشرفته و به ترتیب اولویت ارزش کاربردی)

### ۱. دستیار هوشمند سلامت شغلی (ادغام با هوش مصنوعی Gemini)
*   **تحلیل هوشمند پرونده‌ها:** مجهز به موتور پردازشی مدل `gemini-3-flash-preview` از طریق SDK مدرن `@google/genai` برای تحلیل و تفسیر آنی داده‌های ادیومتری، اسپیرومتری و سوابق بالینی.
*   **مشاوره و پشتیبانی هوشمند:** چت‌بات تعاملی درون‌برنامه‌ای به زبان فارسی برای کمک به پزشکان طب کار و افسران سلامت جهت جستجو و تطبیق شرایط بالینی بیمار با استانداردهای ملی و بین‌المللی سلامت کار (OSHA و HSE).

### ۲. پایش دقیق و چندبعدی سلامت و پاراکلینیک
*   **ماژول پیشرفته شنوایی‌سنجی (Audiometry):** ثبت و رسم نمودارهای سنجش شنوایی (Air Conduction) در فرکانس‌های مختلف به همراه تست‌های سنجش گفتار (SRT و SDS) و تولید خودکار گزارش تخصصی ادیولوژی.
*   **ماژول پیشرفته تنفس‌سنجی (Spirometry):** پایش حجم‌های ریوی از جمله FVC، FEV1 و PEF با قابلیت تفسیر خودکار الگوهای انسدادی (Obstructive)، انسدادی-انقباضی (Mixed) و انقباضی (Restrictive).
*   **بینایی‌سنجی (Vision) و تست‌های آزمایشگاهی:** ثبت وضعیت حدت بینایی (با عینک و بدون عینک)، وضعیت دید رنگ، دید عمق و میدان بینایی در کنار پنل جامع واردکردن نتایج بیوشیمی و خون‌شناسی (FBS، Lipid Profile، CBC و آنزیم‌های کبدی).

### ۳. مدیریت گردش کار و ارجاعات دوره‌ای (پزشک، کارشناس و پیگیری)
*   **کنترل دسترسی مبتنی بر نقش (RBAC):** وجود پنل‌های اختصاصی و متمایز برای «پزشک طب کار»، «افسر سلامت/بهداشت حرفه‌ای»، «مدیریت سازمان» و «توسعه‌دهنده».
*   **چرخه ارجاع هوشمند:** فرآیند ساختاریافته ارجاع پرونده‌ها در وضعیت‌های مختلف (`waiting_for_doctor` و `pending_specialist_result`) و ثبت مستقیم نتایج مشاوره تخصصی (Specialist Follow-up).

### ۴. معماری آفلاین-اول با ذخیره‌سازی رمزنگاری‌شده محلی (Crypto-Secure Storage)
*   **امنیت در سطح کلینیک:** ذخیره‌سازی پرونده‌های پزشکی بیماران به صورت کاملاً رمزشده در حافظه محلی سیستم پیش از ذخیره در LocalStorage با استفاده از الگوریتم سفارشی XOR Cipher و تکنیک سریال‌سازی Base64 UTF-8 جهت حداکثر وفاداری به کاراکترهای فارسی و جلوگیری از دسترسی‌های غیرمجاز.
*   **پشتیبان‌گیری و بازیابی داده‌ها:** قابلیت خروجی گرفتن و بازیابی فایل‌های پشتیان کامل به فرمت JSON برای جا‌به‌جایی امن سوابق.

### ۵. گزارش‌گیری پویا و برون‌بری اکسل
*   **گزارش‌گیری سازمانی Excel:** تبدیل آنی لیست کارکنان و آخرین وضعیت معاینات بالینی و پاراکلینیکی به جداول استاندارد اکسل به همراه قالب‌بندی خودکار ستون‌ها بر اساس محتوای فارسی.
*   **ورود اطلاعات تاریخی (Historical Import):** تبدیل و واردکردن سریع داده‌های آرشیوی سازمان‌ها از اکسل به سیستم با نقشه‌برداری هوشمند فیلدهای فارسی بدون نیاز به ثبت مجدد.

### ۶. داشبورد مدیریتی و هوش تجاری مدرن (Dynamic Analytics)
*   **نمودارهای تحلیلی تعاملی:** مصورسازی نرخ شیوع اختلالات بینایی، شنوایی، ریوی و بیماری‌های متابولیک با استفاده از فریم‌ورک قدرتمند `Recharts`.
*   **سیستم رفت‌وبرگشت به جزییات (Drill-down):** امکان کلیک روی بخش‌های مختلف نمودار برای باز شدن مدال‌های پویا و مشاهده مستقیم لیست پرسنلی که دارای عارضه مربوطه هستند.

## 💻 تکنولوژی‌های مورد استفاده

*   **فرانت‌اند و فریم‌ورک اصلی:** React 19 (کتابخانه کامپوننت‌های بهینه‌شده به همراه پردازش واکنش‌گرا)
*   **بستر دسکتاپ:** Electron v26 (ارائه نرم‌افزار به صورت یک برنامه محلی و تحت دسکتاپ برای سیستم‌عامل‌های ویندوز، مک و لینوکس)
*   **زبان توسعه:** TypeScript (توسعه کاملاً تایپ‌ایمن و پایدار)
*   **موتور هوش مصنوعی:** `@google/genai` (مدل Gemini 3.5 Flash Preview)
*   **ابزار بیلد و باندلینگ:** Vite و PostCSS به همراه Tailwind CSS v3 برای استایل‌دهی مدرن با پشتیبانی کامل از حالت تاریک (Dark Mode) و لایه‌های RTL.
*   **کتابخانه‌های فرعی محاسباتی:**
    *   `recharts` جهت رسم نمودارهای پویا
    *   `xlsx` برای درون‌بری و برون‌بری فایل‌های اکسل
    *   `html2canvas` و `jspdf` برای خروجی گرفتن پرونده‌های کاغذی

## 🛠⚙️ راهنمای استفاده سریع و راه‌اندازی پروژه

برای توسعه و اجرای این پروژه روی سیستم خود، مراحل زیر را دنبال کنید:

### ۱. نصب نیازمندی‌ها
پیش از شروع، مطمئن شوید که [Node.js](https://nodejs.org) روی سیستم شما نصب شده است. سپس وابستگی‌های پروژه را با دستور زیر نصب کنید:

```bash
npm install
```

### ۲. اجرای در حالت توسعه (Development)
برای اجرای نسخه آزمایشی و توسعه نرم‌افزار به صورت دسکتاپ (Electron + Vite):

```bash
npm run dev
```

### ۳. بیلد و خروجی وب (Production Build)
برای ایجاد نسخه بهینه‌سازی‌شده وب برنامه و خروجی در پوشه `dist`:

```bash
npm run build
```

### ۴. بسته‌بندی برنامه دسکتاپ (Packaging Desktop App)
پروژه مجهز به `electron-builder` برای ساخت فایل‌های نصبی پایدار برای سیستم‌عامل‌های مختلف است. برای بسته‌بندی نهایی نسخه دسکتاپ از دستور زیر استفاده کنید:

```bash
npm run electron:build
```
فایل خروجی و نصبی در پوشه `dist-electron` یا مطابق تنظیمات بیلد در فایل `package.json` تولید خواهد شد.
